# ConsoleAppToCopyFilesAndFolderStructure
Console App To Copy Files and Folder Structure using C#

To copy files and folder/ sub folders from directory using c# console app.

eg :
Sorce path :
C:\

ABC folder
-> abc folder
    --> abc1_file1.doc
    --> abc2_file2.pdf
    --> abc3_file3.xlsx
-> xyz folder
    --> xyz_file1.pdf
    --> xyz_file2.docx
    -->xyz_file3.txt
->test.txt

Destination path :
-----Copy to D:\

ABC folder
-> abc folder
    --> abc1_file1.doc
    --> abc2_file2.pdf
    --> abc3_file3.xlsx
-> xyz folder
    --> xyz_file1.pdf
